**Mini Q&A Platform built with Next.js ‚Äî inspired by StackOverflow.**

*QuestStack* is a hands-on project from the "**Next.js Real Project Mini Class**" hosted by **Dunia Coding** ([duniacoding.id](https://duniacoding.id/)).
In this class, I learned how to build modern web applications using Next.js and create a ready-to-develop website framework.

The website is designed as a simple question-and-answer (Q&A) platform ‚Äî where users can:

- Create a question (Question)
- Give an answer (Answer)
- View a list of recent questions
- View question details and answers

The name "*QuestStack*" comes from a combination of the words *Question* and *Stack*, representing a collection of knowledge generated from various user questions - inspired by the concept of **StackOverflow**, but in a lighter and more minimalist version.
